# Douyin-Api
This API group provides access to Douyin, the Chinese version of TikTok and one of the largest short video platforms in the world. These APIs cover core data types including video details, user profiles, search functions, and comment threads. api


Telegram: https://t.me/ksiaoking
Interfaces 可用接口

Video Detail API / 视频详情接口

Retrieve metadata for a video, such as title, description, views, likes, author, hashtags, and music.
获取视频的基础信息，包括标题、描述、作者、标签、音乐、播放数、点赞数等。

User Detail API / 用户详情接口
Get detailed profile info of a Douyin creator including follower stats, verification, bio, etc.
获取抖音用户的详细资料，包括粉丝数、认证状态、签名等。

User Video List API / 用户视频列表接口
Return the list of videos published by a specific user.
查询某个用户所发布的全部视频。

Search Video API / 视频搜索接口
Search videos by keyword or hashtag.
按关键词或话题搜索短视频内容。

Search User API / 用户搜索接口

Locate creators using names or keywords.
使用关键词搜索抖音达人或普通用户。

Video Comment API / 视频评论接口
Retrieve comment threads from a video, including likes and thread structures.
获取视频下的评论内容，包括评论热度、点赞、回复结构等。
